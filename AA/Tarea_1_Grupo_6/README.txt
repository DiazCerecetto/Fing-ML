Laboratorio 1, Aprendizaje automático 2023, grupo 6

Proporcionamos los siguientes archivos:

- Una entrega de Notebook que importa el archivo "lab1.py" y 
sólo realiza las llamadas a funciones en las partes donde es necesario.

- Incluímos además un menú interactivo en el archivo "lab1.py" por si 
querrían comprobar los resultados de las evaluaciones de forma ágil.


**********************************
*********** IMPORTANTE *********** 
**********************************


Las dos entregas modifican el archivo "data.csv", el cual se debe incluir en el mismo
directorio de los notebooks y el código. Este archivo debe CARGARSE NUEVAMENTE si se
quiere ejecutar otra forma de entrega (por ejemplo, ejecutar el notebook
y posteriormente el script "lab1.py" o viceversa), de lo contrario, causará errores.


*****************************************************
*********** EJECUTAR ENTREGABLE DE CODIGO ***********
*****************************************************

Para ejecutar el código del entregable "lab1.py": 

1 - Insertar el archivo data.csv obtenido desde la página web 
proporcionada, en el mismo directorio que el script.

2 - Abrir una terminal en este directorio y ejecutar:

	python lab1.py 

3 - El menú desplegará instrucciones para ejecutar 
cada evaluación distinta indicada en el informe
